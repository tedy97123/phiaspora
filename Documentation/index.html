Phiaspora - Next Js Fashion Store eCommerce Template

Introduction

Phiaspora offers a Fashion Store eCommerce Next Template. Phiaspora is fully responsive and built with Next 14.X, React 18.X and Tailwind css v3.x. E-commerce Template. It is fully responsive on all devices and has multiple layout options. Phiaspora is the only eCommerce Next template you’ll need to create a powerful online store that will give your customers a smooth and seamless shopping experience. Phiaspora comes with a Modern design, a minimalist feel, and a range of special demos, Phiaspora is designed for various online retailers. Display your products in style and make browsing, choosing, and purchasing easy for your customers.

Dark & RTL version Available: We have added a RTL version in both Dark and Light versions.

How can you change the font-family?

Please follow the step by step process

Step 1

First of all open _fonts.scss (path: Phiaspora_Nextjs/src/app/assets/scss/custom/_fonts.scss) file,
and add your font family
@import url('https://fonts.googleapis.com/css2?family=DM+Sans:opsz,wght@9..40,400;9..40,500;9..40,600;9..40,700&display=swap');
Import the @import link from your Google Fonts.

Step 2

Import your font family in layout.js (path: Phiaspora_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Phiaspora_Nextjs/src/app/layout.js) file.

import { Dm_sans } from 'next/font/google' - Themes primary font family is Dm Sans
Step 3

Declare your font family variables in tailwind.config.js file, or replace your font name in tailwind.config.js file.

fontFamily: {
'dm_sans': "var(--font-dm_sans)"
},
- Themes primary font family is Dm Sans
Step 4

Pass your font family in layout.js (path: Phiaspora_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Phiaspora_Nextjs/src/app/layout.js) file.

const dm_sans = Dm_sans({
  subsets: ['latin'],
  weight: ["300", "400", "500", "600", "700", "800", "900"],
  display: "swap",
  variable: "--font-dm_sans",
})

- Themes primary font family is Dm Sans
Step 5

And also add font class in body tag in layout.js (path: Phiaspora_Nextjs/src/app/layout.js) file,
<body className={`${dm_sans.variable} font-dm_sans text-base text-black dark:text-white bg-white dark:bg-slate-900`}>{children}</body>
